# PerformanceAssistant介绍
PerformanceAssistant （PA）是一款性能助手，提供 Linux 系统下的性能分析和性能调优功能。 

- 获取系统信息会获取硬件和操作系统信息。
- 性能分析会自动分析系统性能，并给出一些性能指标、建议，帮助分析、定位系统中存在的性能瓶颈。
- 性能调优会自动优化系统性能，并根据实际情况提供优化建议。

# 获取系统信息

- CPU信息
- 硬盘信息
- 内存信息
- 主板信息
- 网卡信息
- 操作系统信息

# 性能分析

- CPU利用率与负载 
- 硬盘I/O负载 
- 硬盘利用率 
- 硬盘inode利用率 
- 内存利用率 
- 网络连接状态 
- 占用CPU高的前10个进程 
- 占用内存高的前10个进程 
- 查看网络流量
- JVM

# 性能调优功能列表

- Linux内核调优
- JVM性能调优
